小鹅通视频下载
----
此脚本仅供学习参考，如有任何纠纷问题作者不负责（请参考LICENSE）


使用前修改内容
----
APPID：PC版登陆后的URL中有

XIAOEID：COOKIE中XIAOEID

RESOURCEID：某个课程的ID，点击课程后URL最后的id=

sessionid：COOKIE中的laravel_session


使用方式：
----
* Python 3

```
pip install -r requirements.txt

python xiaoet.py
```
